# Selenium Java Automation Framework

A professional test automation framework built with Selenium WebDriver and Java.

## Tech Stack
- Java
- Selenium WebDriver 4.41.0
- TestNG
- Page Object Model (POM)
- ExtentReports
- Maven
- Jenkins CI
- REST Assured (API Testing)

## Framework Features
- Page Object Model for maintainable test structure
- Data Driven Testing with TestNG @DataProvider
- Automated screenshots on test failure
- ExtentReports for visual HTML reporting
- Jenkins CI integration for automated test execution
- REST Assured API testing - GET, POST, DELETE requests

## Test Scenarios
- Login (valid and invalid credentials)
- Create Employee
- Logout

## How to Run
```bash
mvn test
```
